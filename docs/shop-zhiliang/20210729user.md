# 云仓消费条线
- FAQ：消费者升级成为经销商，上级经销商还拿收益吗？
    - 不拿收益
- FAQ：高等级别代理身份，下次拿了低级的数量，是否降级？
    - 只要一次性买过超过量的货物，自动升级对应级别，之后拿货不论多少都是这个级别价格
- FAQ：低等云仓还有10件货，升级高级后，这10件卖出去收益怎么算？
    - 10件货 原本购买成本80元，升级后，这几件货收益按照成本70元算
    - 也就是10件货按照新等级收益来算，激励代理商升级
